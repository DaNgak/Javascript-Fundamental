Inspect element untuk melihat javascriptnya

Pengertian Array

Array merupakan variabel khusus yang dapat menampung lebih dari satu data pada suatu waktu.

Contohnya kita mempunyai daftar item buah-buahan seperti pisang, pepaya, durian dan apel, kita bisa menyimpan semua item tersebut dalam satu variabel, misalnya variabel buah

Cara Inisialisasi Array

Untuk membuat array pada JavaScript, kita bisa menggunakan sintaks di bawah ini.

Sintaks

var nama_array = [item1, item2, item3, ....];

Atau bisa menggunakan 

var nama_array = new Array(item1, item2, item3, ....);

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];

Atau bisa menggunakan 

var mobil = new Array("Avanza", "Agya","Ayla")

Mengubah Data Array

Selain membuat kita juga bisa mengubah data array.

Sintaks

nama_array[key]="newItem";

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
mobil[0]="Fortuner";
console.log(mobil);

Menghitung Jumlah Array

Menghitung jumlah array, hal ini sangat berguna ketika mempunyai banyak elemen array yang tidak memungkinkan untuk dihitung secara manual.

Sintaks

nama_array.length;

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
console.log(mobil.length); // Hasilnya : "3"

Mengakses Data Array

Variabel yang dideklarasikan dalam fungsi JavaScript, menjadi variabel lokal (hanya di fungsi tersebut).

Sintaks

var nama_array = [item1, item2, item3, ....];
console.log(nama_array[keyitem]);

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
console.log(mobil[1]); //Output : "Agya"
console.log(mobil[0]); // Output : "Avanza"
console.log(mobil[mobil.length - 1]); // output : "Ayla"

Mengubah Array menjadi String (.toString)

Kita dapat mengubah array menjadi string dengan cara seperti dibawah

Sintaks

var nama_array = [item1, item2, item3, ....];
nama_array.toString();

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
console.log(mobil.toString()); // Output : "Avanza,Agya,Ayla"

Mengubah Array menjadi String dengan Delimiter(Pemisah Custom)

Apabila kita ingin mengganti pemisah dari defaultnya koma dengan yang lainnya, kita bisa menggunakan sintaks berikut ini.

Sintaks

nama_array.join(" pemisah_yang_diinginkan ");

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
console.log(mobil.join("-")); // Output : "Avanza - Agya - Ayla"

Mengeluarkan data dari Array (.pop)

engeluarkan data terakhir dari array. Konsepnya sama seperti algoritma Stack

Sintaks

nama_array.pop();

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
mobil.pop(); // mengeluarkan "Ayla" dari array
console.log(mobil.join("-")); // cetak array mobil hasilnya "Avanza - Agya"

Memasukkan data ke Array (.push)

Kita bisa memasukkan data ke dalam array dengan menggunakan sintaks di bawah ini.

Sintaks

nama_array.push("new_item");

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
mobil.push("Fortuner"); //Memasukkan "Fortuner" ke array mobil
console.log(mobil.join("-")); // cetak array mobil hasilnya "Avanza - Agya - Ayla - Fortuner"

Menggabungkan dua Array (.concat)

Kita bisa menggabungkan dua array menjadi satu array saja dengan menggunakan sintaks di bawah ini.

Sintaks

var arraygabungan = Array1.concat(Array2);

Contoh

var mobilBudget = ["Avanza", "Agya", "Ayla"];
var mobilMahal = ["Fortuner", "Lamborghini"];
var mobilgabungan = mobilBudget.concat(mobilMahal);
console.log(mobilgabungan.join("-")); // cetak array mobil hasilnya "Avanza - Agya - Ayla - Fortuner - Lamborghini"

Menggabungkan tiga Array (.concat)

Untuk menggabungkan tiga array, kita bisa melihat sintaks dan contoh di bawah ini.

Sintaks

var Arraygabung = array1.concat(array2,array3);

Contoh

var mobilBudget = ["Avanza", "Agya", "Ayla"];
var mobilMahal = ["Fortuner", "Lamborghini"];
var motor = ["Verza", "CB 150R", "MegaPro"];
var kendaraanBermotor = mobilBudget.concat(mobilMahal, motor);
console.log(mobilgabungan.join("-")); // cetak array mobil hasilnya "Avanza - Agya - Ayla - Fortuner - Lamborghini - "Verza" - "CB 150R" - "MegaPro"

Iterasi Data dari Array dengan For

Kita bisa mengakses seluruh array dengan cara looping menggunakan for.

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
for(i = 0 ; i < mobil.length ; i++)
{
    console.log(mobil[i]);
}

Iterasi Data dari Array dengan ForEach (Lebih Recomend)

Foreach artinya kita tidak perlu membuat iterasi looping dari 0 sampai array.length

Contoh

var mobil = ["Avanza", "Agya", "Ayla"];
mobil.forEach(mobils => {
    console.log(mobils);
});

Iterasi menggunakan metode .map

Kita bisa menggunakan metode map() untuk membuat array baru dengan melakukan fungsi pada setiap elemen array.

Metode map() tidak menjalankan fungsi untuk elemen array tanpa nilai.

Metode map() tidak mengubah array asli.

Contoh

var nomor1 = [5, 10, 15, 20];
var nomor2 = nomor1.map(value => {
    return value * 2;
});

console.log(nomor2.join("-")); // cetak array mobil hasilnya "10 - 20 - 30 - 40"
