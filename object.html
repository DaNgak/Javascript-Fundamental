Inspect element untuk melihat javascriptnya

Pengertian Object

Object pada bahasa pemroraman merupakan "abstraksi" dari benda yang ada di dunia nyata. Object dapat digunakan untuk menyimpan data.

Pada JavaScript, object terdiri dari name dan value(variable) atau method(fungsi)

Inisialisasi Object & Property & Method

Untuk penulisan object pada javascript sedikit berbeda dari bahasa pemrograman lain dimana pada javascript untuk menuliskan class tersebut berorientasi pada JSON (Javascript Object Notation) yang mana berupa array tetapi bermakna object

Contoh

var Mobil = 
{
    nama: "Toyota", 
    tipe: "Agya",
    berat: 600,
    warna: "putih",
    maju : function() {
        return "Mobil Melaju";
    },
    mundur : function() {
        return "Mobil Mundur";
    }
}

Mengakses Property dan Method pada Object

Untuk mengakses object tersebut tidak perlu instansiasi object cukup menuliskan nama variabel yang dianggap class dan juga nama method atau propertynya.

Sintaks

nama_object.nama_property;
//mengakses property dari suatu object
nama_object.nama_method();
//mengakses method dari suatu object

Contoh

console.log("Nama Mobil " + Mobil.nama);
console.log("Method Mobil Maju " + Mobil.maju());
